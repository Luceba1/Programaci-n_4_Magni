# Registro de eventos - TP 3

Proyecto realizado con React + TypeScript + TailwindCSS.

## Incluye

- useEffect para persistencia
- localStorage con la clave `participantes`
- componentización
- filtros combinados con lógica AND
- clase `Participante`
- botón Limpiar filtros
- botón Resetear datos
- mensaje "No hay participantes"
- contador dinámico: "Mostrando X de Y participantes"
- misma estética base del TP 2

## Ejecutar

```bash
npm install
npm run dev
```
